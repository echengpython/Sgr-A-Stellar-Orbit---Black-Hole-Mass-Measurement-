# Sgr A* Stellar Orbit (S2) and Black Hole Mass Measurement

**Project:** Fitting the orbit of the star S2 around Sgr A* using Keplerian 2D model and measuring the supermassive black hole mass.

## Project Structure

```
sgrA_star_project/
│
├── data/
│   ├── s2_positions.csv        # Extracted table data
│
├── src/
│   ├── __init__.py
│   ├── load_data.py            # parsing + preprocessing
│   ├── kepler_orbit.py         # orbit equations
│   ├── fit_orbit.py            # wrapper calling scipy.optimize
│   ├── plot_orbit.py           # visualization
│   └── utils.py
│
├── notebooks/
│   ├── exploration.ipynb       # optional initial testing
│
├── results/
│   ├── fit_report.txt
│   └── orbit_plot.png
│
├── requirements.txt
└── README.md
```

## Installation

1. Clone this repository
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. Place your S2 astrometry data in `data/s2_positions.csv`
2. Run the analysis using the modules in `src/`
3. Check results in `results/`

## Data Sources

- Gillessen et al. 2009
- Ghez et al. 2008
- Or use synthetic data generated by the code

## Goals

- Fit the orbit of the star S2 around Sgr A* using Keplerian 2D model
- Measure the supermassive black hole mass (M_BH) and estimate uncertainties
- Provide visualization, residual analysis, and GitHub-ready project structure
